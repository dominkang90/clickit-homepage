1. 관리자 열쇠(토큰) 등록

깃헙에서 발급한 토큰을 한 번만 붙여넣으면 이 브라우저에 저장됩니다. 토큰이 곧 관리자 비밀번호예요 — 다른 사람에게 보여주지 마세요.

발급 방법: github.com/settings → 토큰 만들기 → Repository access에서 clickit-homepage만 선택 → Permissions에서 Contents: Read and write → Generate. (만료 기간은 1년 추천)
